# Chrome Extension

Versao enxuta do Profit Navigator 2026 para carregar como extensao `unpacked`, agora usando `chrome.sidePanel`.

## Como usar

1. Abra `chrome://extensions`
2. Ative `Developer mode`
3. Clique em `Load unpacked`
4. Selecione a pasta `chrome-extension`
5. Clique no icone da extensao para abrir o side panel

## O que entrega

- Calculo rapido de precificacao
- Rateio do DAS do MEI por venda
- Simples Nacional e Lucro Presumido
- Calculo simplificado de base CBS/IBS e IS
- Leitura de chave NF-e
- Atalho visual para leitura da API piloto da Receita Federal
